Анонимный чат

  1. Каждый раз генерируется новый пользователь
  2. Нет ников, генерируется случайная аватарка
  3. Комната исчезает, когда выходит последний пользователь
  4. Не хранятся личные данные пользователя
  5. Анонимность в первую очередь

Технологии

Штуки, которые мы использовали

  1. Webpack
  2. React
  3. Redux
  4. Socket.io
  5. Express
  6. MongoDB, Mongoose
  7. Docker

Почему именно эти технологии? #1

  • webpack - хотели сборку с хотрелоадом и модулями
  • react и redux - хотели поробовать новые для нас технологии, модно быть в тренде :)
  • socket.io - быстрое внедрение и удобный транспорт с реконнектом и фоллбеками

Почему именно эти технологии? #2

  • express - также быстрое развертывание, простая работа с роутами
  • MongoDB и mongoose - наиболее знакомая технология, тем не менее есть свои подводные камни
  • Docker - хотелось попробовать, в процессе знакомства вы выяснили, что это отличный способ (потратить время) отвязаться от окружения, быстро масштабировать и переносить сервис
